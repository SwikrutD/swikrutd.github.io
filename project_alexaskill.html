Bhagavad Gita Guide – Alexa Skill

Project Overview

A voice-interactive Amazon Alexa skill that delivers 1,000+ quotes and facts from the Bhagavad Gita. Built on the Alexa Developer Platform with an AWS Lambda backend, it supports multiple interaction types and is optimized for natural conversational speech flow using SSML. The skill is publicly listed on the Alexa Skills Store.

June 2025 – August 2025

Key Features

Voice Interaction

Supports 3+ interaction types including random quotes, chapter-specific facts, and thematic teachings from the Bhagavad Gita.

AWS Lambda Backend

Serverless Node.js backend on AWS Lambda, integrated with the Alexa SDK for efficient, scalable request handling.

Published on Alexa Skills Store

Fully published with invocation phrases, publishing assets, and metadata managed through the Alexa Developer Console.

Tech & Tools

  • Amazon Alexa Developer Platform – skill framework and intent handling
  • AWS Lambda – serverless Node.js backend
  • Alexa SDK – request/response lifecycle management
  • SSML (Speech Synthesis Markup Language) – optimized voice output
  • Alexa Developer Console – publishing, metadata, and invocation management

Highlights
